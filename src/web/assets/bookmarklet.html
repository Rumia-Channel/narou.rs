Narou.rs

ブックマークレット

下のリンクをブックマークバーへドラッグして登録します。なろう・カクヨム・小説家等のページでブックマークレットを開くと、表示中の小説 URL をこの WEB UI に持ち帰り、登録確認のダイアログを出します。

Narou.rs に登録

ブックマークレットは現在のタブから narou.rs を別ウィンドウで開き、表示中のページ URL を ?register=... パラメータとして渡します。承認時に既存の /api/download を same-origin で呼び出すため、クロスオリジン CSRF ガードの影響を受けません。

使い方

  • なろう/カクヨム/Arcadia 等の小説ページを開く
  • 登録したブックマークレットをクリック
  • 別ウィンドウで開いた WEB UI の確認ダイアログで「登録」を押す
  • ダウンロードキューに追加され、通常通り処理されます

古いブックマークレット（クロスオリジン POST 用）をお使いの場合は、WEB API の CSRF ガードにより動作しないため、上記のリンクから再登録してください。
