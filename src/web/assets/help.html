Narou.rs

narou.rs WEB UI ヘルプ

概要

narou.rs は小説家になろう・カクヨム等の Web 小説を管理・変換するツールです。この WEB UI からダウンロード・更新・変換・送信等の操作を行えます。

キーボードショートカット

キー動作
Dダウンロード
U更新
C変換
S送信
A全選択
Esc選択解除
Fフィルター入力にフォーカス
?このヘルプを表示

基本操作

  • ダウンロード: URLを入力して小説をダウンロードします
  • 更新: 選択した小説または全小説の更新を確認します
  • 変換: 選択した小説を電子書籍形式に変換します
  • 送信: 変換済みファイルを端末に送信します
  • 凍結: 自動更新の対象外にします

右クリックメニュー

小説一覧の行を右クリックすると、その小説に対する個別操作メニューが表示されます。

タグ

小説にタグを付けて分類できます。タグ名を右クリックすると色を変更できます。
